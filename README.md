# PLANT

PLANT è un vaso intelligente dal design minimale ed elegante, con un comparto smart avanzato per il supporto e la cura del verde domestico. Integrato con Next.js e Supabase.

## Funzionalità

- **Ecosistema Smart**: Sensori integrati per umidità, temperatura, e pH. Irrigazione ottimizzata.
- **Shop Web Moderno**: Store integrato con Supabase, gestione ordini, wishlist (preferiti).
- **Gestione Account**: Dashboard utente personalizzata e gestione dei task domestici e botanici.
- **Design & UX**: Interfaccia "glassmorphic" minimale con supporto Light/Dark mode.
- **SSR & SEO ottimale**: Metadata API globale tramite stack router moderno.

## Tecnologie

- **Core Framework**: React (Next.js - App Router)
- **Database & Authentication**: Supabase (PostgreSQL & Auth)
- **UI & Styling**: Vanilla CSS Globale con Design System predefinito
- **Iconografia**: Lucide Icons

## Getting Started

### Prerequisiti

- Node.js (v18+)
- Database Supabase configurato (.env.local)

### Installazione

1. Clona la repository:
   ```bash
   git clone <repository-url>
   cd PLANT
   ```

2. Installa le librerie:
   ```bash
   npm install
   ```

3. Crea un file `.env.local` con le variabili di Supabase necessarie per il corretto funzionamento.

### Sviluppo

Start development server (Next.js):
```bash
npm run dev
```

Aprire [http://localhost:3000](http://localhost:3000) col browser desiderato.
